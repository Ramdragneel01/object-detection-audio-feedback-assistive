# Changelog

All notable changes to this project are documented in this file.

## [0.2.0] - 2026-04-24

### Added

1. Optional API key auth for protected inference endpoint via `ASSISTIVE_API_KEY`.
2. In-memory request throttling via `ASSISTIVE_RATE_LIMIT_PER_MINUTE`.
3. API integration tests for health, auth, and rate-limit behavior.
4. Production governance baseline files and documentation pack.
5. CI and release workflows for automated quality and release checks.
